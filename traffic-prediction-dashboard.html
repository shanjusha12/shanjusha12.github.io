Citywide Traffic Prediction Dashboard

Project Type: Academic Collaboration

Focus: Data Visualization, Smart Cities

Overview

A dashboard interface to visualize real-time and predictive traffic patterns using spatiotemporal neural networks, created to assist urban planners and commuters.

Role

UI Designer & Data Visualizer — responsible for turning complex data into digestible, visual insights.

Design Approach

  • Collaborated with a backend ML team to understand model outputs and constraints.
  • Created dashboard layouts that emphasized clarity, color contrast, and actionable insights.
  • Designed map-based and tabular views to support different user types.

Tools Used

Figma · Tableau · Python · Illustrator

Key Outcomes

  • Delivered a multi-layered, responsive dashboard with live data capability.
  • Simplified traffic prediction insights for non-technical users.
  • Supported future scalability through modular UI patterns.

Skills Applied

Dashboard UX · Data Visualization · Collaborative Design · Color Mapping · Urban Tech
