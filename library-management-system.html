Library Management System

Project Type: Academic

Focus: UI/UX for Educational Tools

Overview

A web-based interface to manage digital library systems, designed for ease of use across student and administrative roles.

Role

UI/UX Designer — led the redesign of the interface from wireframes to high-fidelity mockups.

Design Approach

  • Identified pain points in legacy flows and proposed simplified navigational structures.
  • Focused on making key features like search, renewals, and requests accessible in one click.
  • Created a modular, card-based design for easier information scanning.

Tools Used

Figma · Adobe XD · Accessibility Tools

Key Outcomes

  • Improved ease of use and reduced cognitive load in core tasks.
  • Built a responsive layout compatible with desktops, tablets, and mobile.
  • Established reusable components to streamline future development.

Skills Applied

Information Architecture · Wireframing · Mobile-First Design · UX Research · Design Systems
