Weight Loss Management Website

Project Type: Personal/Academic

Focus: Health & Wellness, Responsive UI Design

Overview

A responsive web platform designed to support users in tracking wellness goals, improving engagement, and promoting a positive, motivating experience.

Role

Tools Used

Figma · Illustrator · Adobe Color

Key Outcomes

  • Delivered a cohesive, brand-aligned experience across web and mobile.
  • Applied behavioral design patterns to encourage consistency in user activity.

Skills Applied

Responsive Web Design · Visual Branding · UI Prototyping · Typography · Accessibility
